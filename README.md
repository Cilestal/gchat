# gchat
Simple Java NIO client-server chat<br>
<br>
## Build projects
To build the entire Gradle project, you should run the following in the root of the checkout.<br>
```./gradlew build```<br>
<b>Build server</b><br>
```./gradlew server:build```<br>
<b>Build client</b><br>
```./gradlew client:build```<br>
